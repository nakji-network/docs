Solana NFT Data

note

This endpoint is in pre-release and is subjected to change on official launch.

Solana NFT Data is designed to provide all Solana NFT data that lives on the chain. Currently, Nakji offers NFT transaction data by NFT Address and wallet address.

All REST API endpoints require authentication during the private beta.

Base URL: https://api.nakji.network/v1

Endpointsâ€‹

/solnfts/trades/nft/:nftAddressâ€‹

Given a Solana NFT address, displays all trades for that Solana NFT.

Request varsâ€‹

0 GET variables. nftAddress is the address to a Solana NFT. Results are in a descending order by timestamp (ts).

Response varsâ€‹

  • Ts: Timestamp of when the transaction occurred
  • NftAddress: The address of the Solana NFT
  • Transaction: The address of the transaction
  • Buyer: The wallet address of the recipient of the NFT
  • Seller: The wallet address of the sender of the NFT
  • Price: The price the NFT was sold at. The price is displayed in lamports.
  • SoldOn: The marketplace that the NFT transaction took place.
